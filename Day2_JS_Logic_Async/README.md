## 🚀 Key Takeaways
- Use `let` and `const`, not `var`.
- Understand the **Event Loop** to write better async code.
- Always handle errors in `fetch` using `try...catch`.
- **AI Tipped**: Use Antigravity to generate boilerplates but always read the code to understand the logic!
