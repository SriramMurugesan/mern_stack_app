# 🚀 MERN Stack Mastery Curriculum

Welcome to the **MERN Stack Mastery** course! This repository contains a day-by-day structured learning path designed to take beginners from zero to building full-stack applications.

## 📅 Curriculum Overview

### [Day 1: TheVibe Core & Modern Web](Day1_TheVibe_Core/)
- **Focus**: IDE Mastery, AI-assisted development, and HTML/CSS fundamentals.
- **Project**: A high-conversion landing page built with prompt engineering.

### [Day 2: JS Logic & Async Mastery](Day2_JS_Logic_Async/)
- **Focus**: Functional programming, DOM manipulation, Promises, and APIs.
- **Project**: A real-time Currency Converter.

### [Day 3: React Foundations & State Management](Day3_React_State_Management/)
- **Focus**: JSX, Components, Props, `useState`, and `useEffect`.
- **Project**: Dynamic Movie Search App.

## 🛠️ Global Setup

To run any React-based project in this curriculum:

1. **Install Node.js**: Ensure you have [Node.js](https://nodejs.org/) installed.
2. **Navigate to the lab directory**:
   ```bash
   cd DayX_FolderName/lab_project_name
   ```
3. **Install Dependencies**:
   ```bash
   npm install
   ```
4. **Run the Project**:
   ```bash
   npm run dev
   ```
